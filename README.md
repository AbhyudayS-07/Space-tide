# 🌊 Space Tide Language Documentation

## What is Space Tide?

Space Tide is a constructed language made with the goal of creating a language that is simple, beautiful, and logical.

The idea behind Space Tide is to create a language where people do not need to memorize thousands of unrelated words. Instead, a small number of root words can create many meanings through a clear and understandable system.

Space Tide focuses on:

    i. Beautiful design
    ii. Simple learning process
    iii. Maximum 100 root words
    iv. Understandable logic
    v. No exceptions in grammar

The main philosophy of Space Tide is:

"Simple roots. Logical growth. Unlimited expression."

---

# Rules

    i. Words should have easy formation.

    ii. Same words with different meanings should be avoided.

    iii. Double letter words are allowed.

    iv. Words should be easy to pronounce.

    v. Adding 'ip' to a word makes it plural. (as a prefix)

            Example: I - me                                    |      You - tu
                     We (group of self) - mip                  |      They (group of you) - tip


    vi. Adding 'ir' indicates possession.

            Example: My - mir                                  |      Their - tir


    vii. Adding 'b' indicates bigger form (as a prefix).

            Example: Water - driker                            |      Walk - gage
                     Ocean / Sea - briker                     |      Travel - bage


    viii. Adding 'm' indicates medium form (as a prefix).

            Example: Water - driker
                     River - mriker


    ix. Adding 'il' indicates future (as a suffix).

            Example: Today - now
                     Tomorrow - noil


    x. Adding 'el' indicates past (as a suffix).

            Example: Today - now
                     Yesterday - noel


    xi. Adding 'na' indicates opposite/negative (as a suffix).

            Example: Light - suni
                     Dark - nasuni


    xii. Adding 'et' indicates place (as a suffix).

            Example: Live - liv
                     Home (place to live) - livet


    xiii. Adding 'a' to a word indicates part (as a suffix).

            Example: Home - livet
                     Room - liveta

# How Space Tide Works

Space Tide uses a small number of root words and combines them with fixed rules to create new meanings.

Instead of making a completely new word for every idea, Space Tide grows naturally by modifying existing words.

For example:

A root word can change depending on:
- Number
- Ownership
- Size
- Time
- Meaning relation
- Location
- Parts

This allows the language to stay small while still being expressive.

---

# Goal of Space Tide

Space Tide is not made to replace existing languages.

It is an experiment to explore:

    - How simple can a language become?
    - How much meaning can be created from a small vocabulary?
    - Can a language be both logical and beautiful?

The goal is to create a language that feels natural, easy, and enjoyable for anyone to learn.

# Current Status

Space Tide is an evolving constructed language.

Future development includes:

    - Expanding root words
    - Creating sentence structures
    - Developing pronunciation rules
    - Building example conversations
    - Creating a complete dictionary

Space Tide is built on one idea:

"Less memorization. More understanding."
